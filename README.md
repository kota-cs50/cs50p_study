## 内容
- CS50PのWeek 0、1の課題を中心に実施
- Pythonコードに日本語のコメントを追加し、処理内容や分岐の意図を明確にしています
- 学習を振り返るための自分用のメモですが、他の学習者にも参考になることを意識しています

## 使用技術
